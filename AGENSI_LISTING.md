## Description (paste into the description field)

Large data teams spend days writing documentation that nobody keeps up to date.
SnowDoc fixes that.

Paste any SQL file or Snowflake table schema and get instant, publication-ready
documentation — column dictionaries, data lineage maps, CTE breakdowns, business
summaries, and data quality notes — formatted for Confluence or SharePoint.

**What you get:**
- Business summary written in plain English (not engineer-speak)
- Full column dictionary with types, nullability, and descriptions inferred from code
- Data lineage showing source tables → transformations → target
- CTE breakdown with dependency chain
- Data quality flags (risky joins, null assumptions, deduplication gaps)
- Output in Markdown, Confluence format, or SharePoint-ready structure

**Who this is for:**
- Data engineers who need to document pipelines fast
- Analytics engineers maintaining dbt models
- Data teams onboarding new members who need to understand existing SQL
- Anyone who has ever been asked "what does this table mean?" at 4pm on a Friday

**Built by a Senior Snowflake Developer** who spent years writing this documentation
manually and got tired of it.

Compatible with Claude Code, Cursor, and any agent that supports SKILL.md.
